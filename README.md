## Basic Commands

Init project baru

```bash
git init
git remote add [link github]
git add .
git commit -m "[commit message]"
git push origin master
```

Push perubahan ( file baru, ubah file, hapus file, dll )

```bash
git add .
git commit -m "[commit message]"
git push origin master
```
